# Amba iOS SDK

The Amba iOS SDK (`Amba`) for Swift Package Manager.

## Install

Add to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/layers/amba-sdk-ios.git", from: "0.1.0"),
]
```

Or in Xcode: **File → Add Packages → Search Package URL → `https://github.com/layers/amba-sdk-ios.git`**.
